# Environment Variables

## Optional Variables
+ NEXTAUTH_SECRET: A secret key used to encrypt session data.
+ NEXT_PUBLIC_NETWORK: The Web3 network to use.
+ REOWN_PROJECT_ID: The unique identifier for a Reown project.

## Required Variables

### Theme Configuration
+ THEME_NAME: The name of the theme to apply.
+ THEME_POSTS_PER_PAGE: The number of posts displayed per page.

### Web3 Configuration
+ WEB3_SPACE_ADDRESS: The address of your space on the Web3 network.
+ WEB3_SPACE_CREATOR_ADDRESS: The address of the space creator on the Web3 network.

### Storage Configuration
+ STORAGE_BLOG_AUTH_TOKEN: The token used for authentication with the blog storage.
+ STORAGE_DATABASE_URL: The connection URL for the PostgreSQL database.

### Backup Configuration
+ BACKUP_GOOGLE_CLIENT_ID: The client ID for the Google Drive backup service.
+ BACKUP_GOOGLE_CLIENT_SECRET: The client secret for the Google Drive backup service.
